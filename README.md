# Laborat-rio-Main-in-the-Middle
Criei um Malware para uma experiência de invasão e detecção de dados. Usando Python no VScode.

Criei Malware Ransoware, do qual, rouba dados e criptografa, solicitando assim recompensa por sua devolução (descriptografar), sem nenhuma garantia de sua devolução.

Criei também um Malware Keyloggers, do qual, se infiltra dentro do sistema e consegue vizualizar em tempo real tudo o que se é digitado, sem ao menos o usuário saber que esta sendo observado.
